# Изображения

`index.html` ссылается на файлы ниже. Положи сюда оригиналы под этими именами
(формат важен — он зашит в путь). Если хочешь другие имена/форматы — поменяй
пути в `index.html` (поиск по `assets/images/`).

| Файл                        | Где используется                          | Формат |
|-----------------------------|-------------------------------------------|--------|
| `logo.svg`                  | логотип в шапке и подвале + favicon       | SVG    |
| `screen-home.png`           | HERO — «Главный экран»                     | PNG    |
| `screen-pets.png`           | секция «Питомцы и профили»                 | PNG    |
| `screen-health.png`         | секция «Здоровье и трекинг»                | PNG    |
| `screen-ai.png`             | секция «ИИ-ассистент»                      | PNG    |
| `store-appstore.svg`        | бейдж «Скоро в App Store» (HERO + CTA)     | SVG    |
| `store-googleplay.png`      | бейдж «Скоро в Google Play» (HERO + CTA)   | PNG    |
| `store-rustore.svg`         | бейдж «Скоро в RuStore» (HERO + CTA)       | SVG    |

> Все эти ассеты есть внутри исходного бандла `original-bundle.html`
> (зашиты как base64). При необходимости их можно оттуда извлечь.
